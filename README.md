# Deploy-TaskScore

Plataforma corporativa de gestão de tarefas e produtividade para equipes de TI ou outros setores.

O TaskScore Pro foi desenvolvido com foco em controle operacional, SLA, produtividade e organização visual utilizando um sistema Kanban moderno e responsivo.


#Funcionalidades

## Autenticação
- Login corporativo com JWT
- Persistência de sessão
- Controle de usuário

## Gestão de tarefas
- Criar tarefas
- Editar tarefas
- Excluir tarefas
- Atualizar status
- Histórico de alterações

##Kanban interativo
- Drag and Drop
- Organização por status
- Atualização visual dinâmica

## SLA e produtividade
- Controle de prazo
- Pontuação automática
- Indicadores de atraso
- Dashboard operacional

## Gestão de técnicos
- Cadastro de técnicos
- Cargo/função
- Edição e exclusão

## Notificações inteligentes
- Tarefas atrasadas
- Vencimento próximo
- Alertas automáticos

## Interface moderna
- Dark mode
- Responsividade mobile
- UI corporativa premium
